# SPI-master-driver-for-ESP32

This project contains the driver used on a Lolin32 V1.0.0 board to communicate with a ADXL345 accelerometer so that it could monitor and display the acceleration on the X, Y and Z axis.

The code was made with the ESP_IDF software develompent kit.
